# Hazarda en Ruby

A random number game implemented in Ruby.

# Running It

If you use nix

```bash
git clone git@github.com:frectonz/hazarda_en_ruby.git
nix develop --impure
./hazarda.rb
```

or if you have ruby installed

```bash
git clone git@github.com:frectonz/hazarda_en_ruby.git
./hazarda.rb
```
